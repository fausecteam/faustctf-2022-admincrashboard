About AdminCrashBoard

Thid awesome Dashboard allows you and your administrators to manage common tasks easily in a webbrowser.

Usage

After loging in, the Crashbard will automatically scan your home directory for XML-based ".button" files. You can run them in order to execute common tasks, like restarting a service.

Demo

This Demo allows to request an account with immediate creation. deactivate this feature on a productive environment (don't in a CTF, because the system will be stated down otherwise).

Security

This servise is built with security in mind. Hence the authentication uses the ssh configuration of your machine.
